# Lecture 03 — Advanced User Interface Components

This lecture covers advanced UI patterns in the Android Framework UI Toolkit. Students learn to build polished interfaces with Material Design components, navigate between screens using Fragments and the Navigation component, display feedback with Toast and Snackbar messages, and present collections of data with `ListView` and custom adapters.

## Apps

| App | Description | UI Framework |
|-----|-------------|--------------|
| [03-1_MaterialDesign](./03-1_MaterialDesign) | Demonstrates `TopAppBar`, `BottomAppBar`, and Material Design theming | XML Layouts + View Binding |
| [03-2_Fragments](./03-2_Fragments) | Multi-screen app using Fragments, Navigation component, and `BottomNavigationView` | XML Layouts + View Binding |
| [03-3_PopUpMessages](./03-3_PopUpMessages) | Shows how to display `Toast` and `Snackbar` messages with Kotlin extensions | XML Layouts + View Binding |
| [03-4_ListView](./03-4_ListView) | Displays a scrollable list using `ListView` with a custom `ArrayAdapter` | XML Layouts + View Binding |

## Key Concepts Covered

- Material Design components: `TopAppBar`, `BottomAppBar`, menus, and theming
- The Fragment lifecycle and its relationship to the host Activity
- AndroidX Navigation component: `NavHostFragment`, `NavController`, `AppBarConfiguration`
- Bottom navigation with `BottomNavigationView` and `NavigationUI`
- Portrait and landscape layout handling
- `Toast` and `Snackbar` for user feedback
- Kotlin extension functions for reusable UI utilities
- `ListView` and custom `ArrayAdapter` for list-based data display
- Fragment View Binding delegation pattern
